# snake-game-python-OOP
This project is a classic **Snake Game** built while learning **Python and object-oriented programming (OOP)** concepts.

The goal of this project was to understand how larger programs can be structured using multiple classes and modules, while also implementing the core mechanics of a simple game.

## Concepts Practiced

- Object-Oriented Programming in Python
- Working with classes and objects
- Modular program structure
- Game loop logic
- Collision detection
- Score tracking

## Project Structure
main.py → Main game loop and screen setup
snake.py → Snake movement and body logic
food.py → Food generation and placement
scoreboard.py → Score tracking and display

## Features

- Snake movement using keyboard controls
- Random food generation
- Score tracking system
- Collision detection with food and boundaries
- Game over detection
